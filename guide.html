طريقة اللعب والرفع على GitHub

كل ما تحتاجه لتشغيل الموقع محليًا أو رفعه على GitHub Pages.

طريقة اللعب

  1. افتح صفحة القضايا واختر قضية مناسبة للمستوى.
  2. اقرأ التبويبات: الملف، الشهادات، الأدلة، والدافع.
  3. أضف الأدلة المهمة للوحة الأدلة.
  4. استخدم الهنتات إذا احتجت، ثم أرسل التقرير.

طريقة الرفع

  1. فك ضغط الملف ZIP.
  2. افتح GitHub وأنشئ Repository جديد.
  3. ارفع محتويات المجلد نفسها، وليس المجلد المضغوط.
  4. تأكد أن index.html في Root.
  5. من Settings ثم Pages اختر Branch: main و Folder: /root.

ملاحظات تقنية

  • لا توجد مكتبات خارجية.
  • كل الصور SVG داخلية.
  • التقدم محفوظ في LocalStorage.
  • الموقع Responsive ويعمل على الموبايل.
