دليل الاستخدام والرفع

هذه النسخة معمولة خصيصًا حتى تعمل على GitHub Pages بدون مشاكل CSS أو JavaScript.

طريقة اللعب

افتح القضية، اقرأ التبويبات، أضف الأدلة، ثم اختر الحل. لن يسمح الموقع بإرسال التقرير قبل جمع الحد الأدنى من الأدلة.

حفظ التقدم

التقدم يحفظ تلقائيًا في LocalStorage: القضايا المحلولة، الأدلة، والهنتات المستخدمة.

رفع GitHub

ارفع محتويات المجلد مباشرة بحيث يكون index.html في Root للـ Repository وليس داخل فولدر إضافي.

تقدمك الحالي

اختبار الملفات

لو الصفحة ظهرت بتصميم كامل، إذن CSS وJS يعملان. لو ظهرت كنص فقط، تأكد أنك رفعت مجلد css ومجلد js بجانب index.html.
